# Dredd (dredd)

Dredd is a language-agnostic, open source HTTP API testing framework that validates API implementations against API Blueprint or OpenAPI (2.0/3.0) description documents.

## APIs

### Dredd

Dredd is a language-agnostic command-line tool for validating an API description document against a backend implementation. It supports API Blueprint, OpenAPI 2.0, and OpenAPI 3.0, and provides hooks for custom test setup and teardown.

**Human URL:** [https://dredd.org](https://dredd.org)
